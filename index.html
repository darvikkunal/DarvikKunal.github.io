Emotion Detection through Text

Emotion can be conveyed in many forms that can be observed, such as facial and gestural expressions, voice, and written text. Modeled and implemented an emotion detection system using a LSTM network with attention mechanism to relax the notion of modelling and predicted the emotion by looking into the hidden states of whole text with accuracy of 93%.

TIC_TAC_TOE

Developed a simple and fun game for 2 players, X and O. It is played on a 3x3 grid. Each player's goal is to make 3 in a row. Typically, X starts first, but in Gametable's Tabletop Tic Tac Toe, Player 1 starts first on the first game and Player 2 starts first on the next game. The starting player continues to alternate from game to game. This helps keep gameplay fair over time.

INTERNET RELAY CHAT PROTOCOL

Developed an internet relay chat protocol using the socket programming concept in JAVA which employs server, client, and chat rooms.Implemented functionalities like group chatting and private texting at the same time without being disconnected from the server. Ability to connect to more than one room with a single client instance. Added a feature to transfer files to another user and a user-friendly UI to make the IRC more readable and easier to use.

PLANT_DISEASE_DETECTION

This project is an approach to the development of plant disease recognition model,based on leaf image classification, by the use of deep convolutional networks. The developed model is able to recognize 38 different types of plant diseases out of of 14 different plants with the ability to distinguish plant leaves from their surroundings.
